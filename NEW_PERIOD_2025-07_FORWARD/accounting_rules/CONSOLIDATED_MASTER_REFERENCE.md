# SECTION 4: SALES METHODOLOGY (5-STEP PROCESS)

## üßÆ THE PROVEN 5-STEP SALES ACCOUNTING PROCESS

### **CORRECTED FOR NEW PERIOD (July 1, 2025 Forward)**

This is the **EXACT SAME LOGIC** from the first 6 months, corrected for the new payment structure with Marginalen Bank.

#### **STEP 1: Record ALL Sales to Staging Account 1948**

**Purpose:** Capture total gross sales (including VAT)

```
#VER "A" X YYYYMMDD "Sales description"
{
#TRANS 1948 {} [GROSS_SALES]     [All sales including VAT]
#TRANS 3051 {} -[NET_SALES]      [Net sales = Gross √∑ 1.25]
#TRANS 2611 {} -[VAT_25%]        [VAT = Gross - Net]
}
```

**Calculation Method:**
```
Gross Sales (from POS) = Known
Net Sales = Gross Sales √∑ 1.25
VAT = Gross Sales - Net Sales

Example: 10,000 SEK gross
Net: 10,000 √∑ 1.25 = 8,000 SEK
VAT: 10,000 - 8,000 = 2,000 SEK (25%)
```

**Includes:**
- EasyCashier POS sales
- Shopify e-commerce sales
- All payment methods combined (card + cash)

#### **STEP 2: Reconcile Worldline Card Payments**

**Purpose:** Move card payments from staging to Worldline account

```
#VER "A" Y YYYYMMDD "Worldline Card Payments"
{
#TRANS 1947 {} [WORLDLINE_AMOUNT]  [Worldline card payments]
#TRANS 1948 {} -[WORLDLINE_AMOUNT] [Remove from staging]
}
```

**Verification:** Worldline total from monthly reports (PDFs)

#### **STEP 3: Reconcile Remaining as Cash Payments**

**Purpose:** Identify cash portion (difference)

```
#VER "A" Z YYYYMMDD "Cash Sales (Kontant f√∂rs√§ljning)"
{
#TRANS 1910 {} [CASH_AMOUNT]     [Remaining = Cash sales]
#TRANS 1948 {} -[CASH_AMOUNT]    [Remove from staging]
}
```

**Calculation:** Cash = Gross Sales Total - Worldline Total

**Note:** This step is ONLY if cash sales exist. If ALL sales are card, skip this.

#### **STEP 4: Transfer Worldline to Marginalen Bank**

**Purpose:** Move Worldline balance to Marginalen (physical bank account)

```
#VER "A" W YYYYMMDD "Worldline to Marginalen Bank Transfer"
{
#TRANS 1930 {} [WORLDLINE_AMOUNT]  [Marginalen receives money]
#TRANS 1947 {} -[WORLDLINE_AMOUNT] [Worldline transfer complete]
}
```

**Why:**
- Worldline terminal processes card payments
- Money physically deposits to Marginalen bank account
- We track through 1947 first, then reconcile to 1930

#### **STEP 5: Verification - All Accounts Balance**

```
‚úÖ Account 1948 (Sales staging) = 0 SEK (all sales reconciled)
‚úÖ Account 1947 (Worldline) = 0 SEK (all transferred)
‚úÖ Account 1910 (Cash) = Cash amount or 0 SEK
‚úÖ Account 1930 (Marginalen) = Worldline card amount
```
